Introduction to Programming is a foundational module designed to introduce students to the core concepts of procedural programming and computational problem solving. The module develops students’ ability to analyse simple problems, design structured solutions, and implement those solutions using basic programming constructs such as sequence, selection, iteration, functions, and data structures.

As a tutor for this module at Dublin City University, responsibilities included supporting students during lectures and labs, helping them understand key programming concepts, and providing guidance as they developed their coding skills through exercises and assignments. The role also involved grading projects and offering constructive feedback to support student progress and improve both technical accuracy and problem-solving approach.

The module places strong emphasis on practical programming competence, with students learning how to read, write, test, and debug code while building confidence in applying fundamental programming techniques. It provides an important starting point for later modules in software development, data structures, and more advanced computational thinking.
